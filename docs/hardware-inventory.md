# BEAN Hardware Inventory

## Compute

- NVIDIA Jetson Orin Nano Super Developer Kit
- Raspberry Pi 4B, 4GB
- Arduino Nano x3

## Vision / audio / UI

- USB webcam with microphone
- USB powered speakers / monitor speakers
- 10-inch Android tablet for possible robot face/display
- Portable monitor / display used during Jetson testing

## Motion and motor control

- 555 motors with gearboxes
- BTS7960 43A H-bridge motor drivers
- Arduino Nano low-level controller layer
- Encoder integration planned

Older notes listed 775 motors, but current corrected build direction uses 555 motors with gearboxes.

## Power

- Milwaukee M12 battery adapters
- DC-DC buck converters
- Dedicated 5V / 5.1V supply path for Pi-class compute
- Fused distribution block / LED blade fuse box with negative bus
- Emergency stop switches

## Servos and expansion

- PCA9685 16-channel servo driver
- MG995 13kg metal gear servos
- Future better servo upgrades likely

## Mechanical / fabrication

- PETG filament
- Brass heat-set threaded insert kit with insert tool
- Stainless metric fastener assortment, M2-M5 range
- R188ZZ bearings
- Neodymium magnet assortment

## Sensors

- Planned four proximity sensors:
  - front-left
  - front-right
  - rear-left
  - rear-right

## LiDAR

LiDAR is intentionally not first priority. Current plan is to get BEAN rolling first, then add low-cost LiDAR or better navigation sensors later.
